# Git Commits

Commit message format: `type(scope): description`

Types: feat, fix, refactor, test, docs, chore, ci, perf
Scopes: search, mcp, frontend, backend, docker, ci, deps, auth, cot

Examples:

- `feat(search): add agentic search mode with tool execution`
- `fix(mcp): handle timeout in gateway client`
- `test(search): add unit tests for pipeline resolution`
- `refactor(backend): extract embedding logic to service`
